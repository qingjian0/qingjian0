# 贡献指南 / Contributing

感谢您对项目的关注！欢迎通过 Issue 和 Pull Request 贡献代码。

## 提交 Issue
- 使用提供的 Issue 模板
- 提供尽可能多的信息

## 提交 PR
1. Fork 项目
2. 创建新分支: git checkout -b feature/your-feature
3. 提交改动: git commit -m "feat: add your feature"
4. 推送分支: git push origin feature/your-feature
5. 打开 Pull Request

## 代码风格
- Python: 遵循 PEP 8
- 注释: 使用中文或英文
- 测试: 新功能应包含相应的测试
